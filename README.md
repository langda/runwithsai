# Run with Sai

Visit: [https://runwithsai.com/](https://runwithsai.com/)

## For Developers

### Setup

1. Install Ruby
2. Install dependencies:

   ```bash
   bundle install
   ```

### Run Locally

Start the server:

```bash
bundle exec jekyll serve
```

For live reload:

```bash
bundle exec jekyll serve --livereload
```

Open [http://localhost:4000](http://localhost:4000) to view the site.
